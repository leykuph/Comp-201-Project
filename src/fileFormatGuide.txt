first line is always the numebr of cities

after every section there must be a line only consisting of "-"

first section after the city count initializes cities with their neighbour sizes
ex: Berlin,3 means we create a city named Berlin with 3 neighbours

after the city creation section, we must have neighbour section
neighbour format is like this:
NAME,DISTANCE
neighbour addition order is in the order which we created the cities in section 2,
so the first created city's neighbours will be initialized first
DISTANCE must be a double value (ex: 500.0)
NAME must match the name in section 2


see CityTest.java file for examples
